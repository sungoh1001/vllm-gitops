Reference 0001

Clean OS → GPU 컨테이너 설치 치트시트

연관 수업: Lesson 0001 · 인쇄용 · 마지막 갱신: 2026-06-17

작업 순서

#단계핵심 명령어검증
1APT 미러sudo vim /etc/apt/sources.list.d/ubuntu.sources
sudo apt update		apt update 성공
2Docker Enginesudo apt install docker-ce docker-ce-cli containerd.io docker-compose-plugin
sudo usermod -aG docker $USER		docker --version
3NVIDIA CTKsudo apt install nvidia-container-toolkitnvidia-ctk --version
4GPU 런타임 등록sudo nvidia-ctk runtime configure --runtime=docker
sudo systemctl restart docker		cat /etc/docker/daemon.json 에 nvidia 런타임 존재
5Private Registry/etc/docker/daemon.json에 "registry-mirrors" 추가
sudo systemctl restart docker		docker info 에 Registry Mirrors 표시
6GPU 컨테이너 테스트docker run --rm --gpus all nvidia/cuda:12.6-base nvidia-smi컨테이너 안에서 A100 출력

핵심 파일

파일역할
/etc/apt/sources.list.d/ubuntu.sourcesAPT 패키지 소스 (Ubuntu 24.04 DEB822 포맷)
/etc/docker/daemon.jsonDocker 데몬 설정 — 런타임, 레지스트리 미러
/etc/nvidia-container-runtime/config.tomlNVIDIA Container Runtime 세부 설정(일반적으로 직접 편집 불필요)

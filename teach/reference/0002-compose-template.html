compose.yaml + run.sh 템플릿

연관 수업: Lesson 0002 · 인쇄용 · 갱신: 2026-06-18

디렉토리 구조

~/vllm-experiments/
├── run.sh              ← 실험 설정 (수정 대상)
├── compose.yaml        ← 컨테이너 정의 (고정)
├── .env                ← HF_TOKEN (Git ❌)
├── .gitignore
├── models/             ← 모델 캐시 (Git ❌)
└── results/            ← 실험 결과

compose.yaml (템플릿)

services:
  vllm-dev:
    image: nexus.internal.company.com/vllm/vllm-openai:v0.7.1
    container_name: vllm-dev
    deploy:
      resources:
        reservations:
          devices:
            - driver: nvidia
              count: 1
              capabilities: [gpu]
    environment:
      - NVIDIA_VISIBLE_DEVICES=0
      - HF_TOKEN=${HF_TOKEN}
      - HF_ENDPOINT=https://hf-mirror.internal.company.com
    ports:
      - "8000:8000"
    volumes:
      - ./models:/root/.cache/huggingface
      - ./results:/workspace/results
    shm_size: "2gb"
    command: >
      --model ${MODEL}
      --tensor-parallel-size ${TP_SIZE}
      --gpu-memory-utilization ${GPU_MEM_UTIL}
      --max-model-len ${MAX_MODEL_LEN}
      --dtype ${DTYPE}
      --host 0.0.0.0
      --port 8000

run.sh (템플릿)

#!/bin/bash
# === 실험 설정 (여기만 수정) ===
MODEL="meta-llama/Llama-3.1-8B-Instruct"
TP_SIZE=1
MAX_MODEL_LEN=8192
GPU_MEM_UTIL=0.90
DTYPE="bfloat16"

# 환경변수 로드
set -a; source .env; set +a

# 실행
docker compose up -d

vLLM 핵심 인자 참고

인자의미권장값
--tensor-parallel-sizeGPU 분할 개수A100 PCIe: 1 (NVLink 없음)
--gpu-memory-utilizationVRAM 사용률0.85 ~ 0.95
--max-model-len최대 컨텍스트 길이8192 → 16384 → 32768 순차 실험
--dtype추론 정밀도bfloat16 (A100 네이티브)
--enforce-eagerCUDA Graph 비활성오류 시에만 추가

자주 쓰는 명령어

docker compose up -d          # 컨테이너 시작 (백그라운드)
docker compose logs -f        # 로그 실시간 확인
docker compose down           # 컨테이너 정지 + 삭제
docker compose restart        # 컨테이너 재시작
curl localhost:8000/v1/models # API 확인
